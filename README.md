# Playwright E2E Automation Portfolio
This repository contains a professional End-to-End (E2E) automation suite for a standard e-commerce platform using **Playwright** and **Node.js**.

## 🚀 Project Overview
The goal of this project was to move beyond basic recording and implement a maintainable, cross-browser testing suite. 

### Key Features:
- **Cross-Browser Testing:** Verified on Chromium, Firefox, and Webkit.
- **Locator Abstraction:** Uses a variable-based approach for clean, readable code.
- **Data-Driven Selectors:** Utilizes `data-test` attributes for high stability.
- **Automated Reporting:** Includes screenshots and trace captures on failure.

## 🛠️ Tech Stack
- **Framework:** Playwright (latest)
- **Language:** JavaScript
- **Test Runner:** Playwright Test

## 📋 How to Run
1. Clone the repo: `git clone <your-repo-link>`
2. Install dependencies: `npm install`
3. Run tests: `npx playwright test`
4. View Report: `npx playwright show-report`
